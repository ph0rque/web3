# Networks

[The city analogy](https://medium.com/dragonfly-research/blockchains-are-cities-564327013f86)
* L1s are cities
* L2s and rollups are skyscrapers
* interops are single-purpose towns for a given city
* City <--> L1 mapping:
  * ETH: NYC
  * Sol: LA
  * Avalanche: Chicago
  * NEAR: San Francisco
